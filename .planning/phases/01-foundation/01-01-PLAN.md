# Plan 01-01: Project Scaffold & Tech Decisions

## Objective
Bootstrap myCelebration som et Next.js App Router-projekt med Tailwind CSS, shadcn/ui og Vercel-deployment. Dokumentér de tekniske beslutninger der gælder for hele projektet.

## Context
- `.planning/PROJECT.md` — projektbeskrivelse og beslutninger
- `.planning/phases/01-foundation/01-CONTEXT.md` — fasekontekst og tech decisions

## Tasks

### Task 1: Bootstrap Next.js projekt
**Action**: Kør `npx create-next-app@latest . --typescript --tailwind --eslint --app --src-dir --import-alias "@/*"` i projektmappen. Verificer at dev-server starter uden fejl med `npm run dev`. Tilpas `next.config.ts` med `images.remotePatterns` for Supabase Storage domain (sæt placeholder indtil Supabase-projekt er oprettet i Plan 02).

**Verify**: `npm run dev` returnerer 200 på `http://localhost:3000`

**Done**: Next.js App Router projekt starter lokalt uden fejl og TypeScript kompilerer rent.

---

### Task 2: Opsæt shadcn/ui + projektstruktur
**Action**: Kør `npx shadcn@latest init` og vælg default style (New York), neutral base color, CSS variables. Installér basis-komponenter der bruges i alle layouts: `button`, `input`, `label`, `separator`, `sheet`, `dropdown-menu`, `toast`. Opret mappestruktur:
```
src/
  app/
    (admin)/admin/         # admin route group
    [uuid]/                # guest + screen routes
  components/
    admin/
    guest/
    screen/
    shared/
  lib/
    supabase/
    auth/
    i18n/
  types/
```

**Verify**: `npx shadcn@latest add button` kører uden fejl. Mappestruktur eksisterer.

**Done**: shadcn/ui initialiseret og basis-mappestruktur oprettet.

---

### Task 3: Link til Vercel + dokumentér tech-beslutninger
**Action**:
1. Kør `vercel link` og link projektet til Vercel (opret nyt projekt hvis nødvendigt). Kør `vercel deploy` for at verificere preview-deployment virker.
2. Opdatér `.planning/PROJECT.md` under "Architecture Influences" med de endelige tech-valg (erstat "ikke fastlagt endnu" med de valgte libraries). Commit alle ændringer med besked `[PHASE-1] Bootstrap Next.js project with Tailwind + shadcn/ui`.

**Verify**: Vercel preview URL returnerer 200. `git log --oneline -1` viser commit-beskeden.

**Done**: Vercel preview kører og tech-beslutninger er dokumenteret og committet.

## Output
- `package.json` med Next.js, Tailwind, shadcn/ui
- `src/` mappestruktur klar til Phase 1 Plan 02 og 03
- Vercel projekt linket og preview-deployment verificeret
- `.planning/PROJECT.md` opdateret med endelige tech-valg
